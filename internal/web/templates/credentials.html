If your D&D Beyond character sheet is set to Public, you only need its character id — no cookie at all. Private sheets need the session cookie below.

1 · Grab the character id (one click)

Drag this button to your bookmarks bar, open your character on D&D Beyond, then click it:

❤️ Send to Health Bar

It reads the character id from the page URL and sends it here. (The login cookie is protected by your browser and can't be read this way — paste it below if needed.)

2 · Or enter details manually

To find it: open D&D Beyond logged in → browser DevTools → Application → Cookies → copy the CobaltSession value (or the whole Cookie header).
